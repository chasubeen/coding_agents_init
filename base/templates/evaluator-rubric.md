## 채점 (각 0~2점)

| # | 차원 | 0 (미흡) | 1 (부분) | 2 (충족) | 점수 |
|---|------|----------|----------|----------|------|
| 1 | **Correctness** — 목표와 일치하는가 | 동작 안 함/목표 벗어남 | 일부만 | 목표대로 동작 | |
| 2 | **Verification** — 증거와 함께 검증됐는가 | 검증 없음 | 부분 검증 | 필요한 검사 + 증거 기록 | |
| 3 | **Scope discipline** — 범위를 지켰는가 | 무관한 곳 대거 변경 | 약간 벗어남 | 기능 범위 내 유지 | |
| 4 | **Reliability** — 재시작 후에도 유지되는가 | 깨짐 | 불안정 | 재현·지속됨 | |
| 5 | **Maintainability** — 이해 가능한가 | 난해/문서 없음 | 보통 | 코드·문서 충분 | |
| 6 | **Handoff readiness** — 저장소만으로 이어갈 수 있는가 | 구두 설명 필요 | 일부 누락 | 저장소만으로 계속 가능 | |
